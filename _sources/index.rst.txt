CircassianDNA Chatbot documentation
===================================

A chatbot for the Circassian DNA Project. It answers questions about DNA testing,
haplogroups, and Circassian ancestry. Runs locally and on AWS Lambda / Render.

.. note::
   Source code and docs: https://github.com/kabartay/circassiandna-chatbot
